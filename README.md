# Dotfiles
> Dotfiles on my Archlinux installation

<!--![](header.png) -->

## Requires
* GNU Stow

## Configs included
* bash
* fish
* local user-dirs.dirs
* x
    * xresources
    * xinitrc

## Installation

```sh
$ git clone https://github.com/Dodicin/dotfiles
```

## Usage

```sh
$ cd .dotfiles
$ stow [config folder]
```
